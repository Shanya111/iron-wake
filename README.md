# iron-wake 🔔

> Telegram-бот для мониторинга USD/JPY — объёмы, зоны маржинальности, алерты.

## Что это

Персональный торговый ассистент. Следит за парой доллар/иена и сообщает в Telegram когда происходит что-то важное: резкий рост объёмов, вход в зону маржинальности, пробой уровня.

## Статус

🌱 Начало — структура проекта создана, разработка впереди.

## Планируемые функции

- [ ] Мониторинг USD/JPY в реальном времени
- [ ] Алерты по объёму (резкий рост/падение)
- [ ] Зоны маржинальности — вход и выход
- [ ] Уведомления в Telegram
- [ ] Расширение на другие пары (в будущем)
