# iron-wake 🔔

> Telegram-бот: алерты на уровни по форекс/металлам/нефти/крипте **и** торговый
> ассистент по крипте (VSA-контекст + сигналы ложного пробоя Spring/Upthrust).

## Что это

1. **Алерты «касание уровня»** — поставь уровень по любому инструменту (10 готовых пар или
   свой тикер Yahoo), бот напишет, когда цена его коснётся. Срабатывание ловится по диапазону
   минутных свечей, направление выбирать не нужно.
2. **Торговый ассистент (крипта)** — контекстный анализ (тренд D1 + сильные/слабые уровни +
   зоны ликвидности по объёму) и авто-сигналы по паттернам **Spring** (лонг) и **Upthrust**
   (шорт) с готовыми вход/стоп/цель. Это подсказки для решений, не авто-торговля.

## Команды

| Команда | Что делает |
|---|---|
| `/start`, `/help`, `/about`, `/privacy` | базовое меню и справка |
| `/alert`, `/myalerts` | поставить алерт на уровень / список и удаление |
| `/analyze [код]` | анализ инструмента: тренд + уровни + зоны (с AI-разбором) |
| `/subscribe` | подписка на торговые сигналы по инструменту |
| `/signals` | последние сигналы |
| `/settings` | пороги движка (объём, глубина пробоя) — для админа |
| `/unsubscribe`, `/myid`, `/pay`, `/broadcast` | прочее (последняя — для админа) |

## Стек

Python · aiogram 3 · SQLite · APScheduler · yfinance (форекс/металлы/нефть) ·
CCXT + pandas/numpy (биржевая крипта с объёмом) · OpenRouter (LLM).

## Как запустить

```bash
pip install -r requirements.txt
cp .env.example .env          # заполни TELEGRAM_BOT_TOKEN, OPENROUTER_API_KEY и др.
python bot.py
```

`bot.db` и `settings.json` создаются автоматически. Тесты движка:

```bash
python tests/test_engine.py
```

**Важно про данные движка:** котировки берутся с публичных эндпоинтов **Kraken** через CCXT
(API-ключи не нужны). Kraken даёт и крипту (BTC/ETH/SOL/TON), и форекс (EUR/USD, GBP/USD, AUD/USD,
USD/CAD, USD/JPY) с реальным объёмом и **не блокирует РФ** — поэтому прокси для движка не нужен.
`CCXT_PROXY` оставлен в коде на случай возврата биржи с гео-блоком. Подробнее о деплое —
[DEPLOY.md](DEPLOY.md), об архитектуре — [CLAUDE.md](CLAUDE.md).
